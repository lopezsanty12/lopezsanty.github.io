# Portafolio de Santiago López

Este es el repositorio del portafolio personal de Santiago López, un Ingeniero de Sistemas apasionado por la automatización y la optimización de procesos.

## Sobre el Proyecto

Este proyecto fue creado para mostrar la experiencia profesional, proyectos y habilidades de Santiago López. El sitio web incluye las siguientes secciones:

- **Inicio:** Una breve introducción y enlaces a perfiles sociales.
- **Acerca de:** Una descripción más detallada sobre Santiago.
- **Experiencia:** Un resumen de su experiencia laboral.
- **Servicios:** Los servicios que ofrece.
- **Portafolio:** Una galería de sus proyectos.
- **Testimonios:** Comentarios de clientes y colegas.
- **Contacto:** Un formulario para ponerse en contacto.

## Tecnologías Utilizadas

- [React](https://reactjs.org/)
- [Vite](https://vitejs.dev/)
- [TypeScript](https://www.typescriptlang.org/)
- [Tailwind CSS](https://tailwindcss.com/) (implícito por las clases de utilidad)

## Cómo Empezar

Sigue estos pasos para tener una copia local del proyecto funcionando.

### Prerrequisitos

Asegúrate de tener [Node.js](https://nodejs.org/) instalado.

### Instalación

1. Clona el repositorio:
   ```sh
   git clone https://github.com/lopezsanty12/santy-lopez-portfolio.git
   ```
2. Instala las dependencias:
   ```sh
   npm install
   ```
3. Inicia el servidor de desarrollo:
   ```sh
   npm run dev
   ```
4. Abre [http://localhost:5173](http://localhost:5173) en tu navegador para ver el resultado.
